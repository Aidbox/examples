# refero-renderer

This package builds a static renderer page (`index.html` + `src/index.tsx`) that mounts Refero and speaks [SDC SMART Web Messaging](https://github.com/brianpos/sdc-smart-web-messaging).

Renderer source: https://github.com/helsenorge/refero

To install dependencies:

```bash
pnpm install
```

To run a dev server:

```bash
pnpm dev
```

To build static assets:

```bash
pnpm build
```
